### Project Manager

O projeto tem como objetivo criar um aplicativo para fazer o CRUD de projetos

## Tecnologia:

 <ul>
  <li>React</li>
  <li>Typescript</li>
  <li>Material ui</li>
  <li>Yup</li>
 </ul>

## Telas:

<ol>
  <li>Login</li>
  <li>Cadastro</li>
  <li>Todos os projetos</li>
  <li>Atualizar projeto</li>
  <li>Criar projeto</li>
</ol>
